JSON

		简单值
		与JavaScript类似，在JSON中也同样可以使用“字符串、数值、布尔值、null”
		但是没有“undefined”，而且JSON中的字符串只能且必须使用双引号。

		对象
		在JavaScript中对象和JSON中对象一样，但是JSON对象属性名必须加双引号，
		使用单引号或者没加引号都会导致语法错误，继而解析失败。
		var js = {name:"张三", sex:"男"};
		var json = '{"name":"张三", "sex":"男"}';

		数组
		JSON数组和JavaScript数组字面量表现方法一致，通过方括号来表示

		new
		1 2 3
		"asdfadf"
		true false
		{} []
