XML

		HTML - 超文本标记语言
		XML - 可扩展标记语言（Extensible Markup Language）
		共同点在于都使用“标记（标签）”来描述内容。

HTML和XML

		HTML
		div p span ul li form table...标签有限，都对应特殊的作用，
		最终都被浏览器解析成网页元素。所以，HTML关注的是“内容展示”。

		XML
		没有标签……它是关注“数据展示”，所有标签都需要自定义，并且没有任何
		特殊作用。

XML语法

  1. 标签名对大小写敏感
  2. 必须也只能有一个根元素（顶级标签）
  3. 所有标签都必须有对应的结束标签（单标签在末尾加斜杠）
  4. 标签属性值都必须加引号
  5. 内容中的空格会被保留
  6. 注释使用
  7. 使用 包含的内容即使是标签也会被当成文本处理

XML标签命名规则

		XML元素
		指的是“开始标签”到“结束标签”的整个部分。
		元素可包含“其它元素”、文本或者两者的混合。元素也可以具有属性。
  1. 名称可以包含字母、数字以及其它字符
  2. 名称不能以数字、标点符号开始
  3. 名称不能以字符“xml”（XML、Xml）开始（只是约定，实际上还是可以用）
  4. 名称不能包含空格
  5. 可使用任何名称，没有保留的字词
  6. 尽量使用“英文字母、数字、下划线”来作为标签名（与JavaScript标识符同规则）
  7. 避免在标签名中使用“- . :”等特殊字符
  8. 如果使用了汉字作为标签名，考虑在不同环境下编码或字符识别所带来的问题

解析与反解析

		解析即将XML类型字符串解析成DOM结构，以达到可以使用DOM操作方法来修改XML
		的目的。
