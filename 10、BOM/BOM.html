BOM

		DOM - document object model
		BOM - browser object model
		浏览器对象模型
		目的在于使用JavaScript与浏览器进行交互

		-------------------------------------------------
		Window 代表浏览器窗口，每个标签都具备一个单独的 Window 实例
		所有JavaScript全局变量、对象、函数均自动成为 window 对象的成员
