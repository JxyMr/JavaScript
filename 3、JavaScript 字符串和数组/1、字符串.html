Unicode 编码和字符相互转换

		charCodeAt(下标)
		返回字符串指定位置的字符的 unicode 编码，这个值是 0~65535 之间的整数

		fromCharCode(Unicode编码)
		将 unicode 编码转换为字符

字符串截取

		substr(start,[length])
		抽取字符串中 start 下标开始的指定个数的子字符串
		start 		开始下标。可以为负数，-1代表最后一个字符，-2指倒数第二个字符
		length 		截取个数，若省略该参数，将一直到字符串末尾截止

		substring(start,[end])
		抽取字符串中指定 start 下标和 end 下标之间的字符，不包含end下标处的字符
		start 		非负整数，开始下标
		end 		非负整数，结束下标，不包含此位置字符。若省略则一直到结尾
