执行顺序

		JavaScript 和 HTML 等都一样，其遵循“从上至下”的规则，“顺序执行”脚本代码。

语句

		JavaScript程序是语句的集合，一条JavaScript语句相当于英语中一个完整的句子。
		一条语句由一个或者多个“表达式、关键字、运算符”组成。
		一条语句写一行，语句之间用分号;隔开，分号是JavaScript语句的终止符。

脚本程序位置

		JavaScript代码必须写在 <script></script>（HTML实体） 中，
		或者以单独的 JavaScript 文件来存储。
		甚至可以将JavaScript代码放在元素的属性值中执行。

		script 标签可以放在网页中的任何位置，但是一般放在 head 标签末尾，或者 body 标签中。
