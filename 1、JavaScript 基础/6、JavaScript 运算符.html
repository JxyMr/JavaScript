算术运算符

		=		赋值运算符
		+-*/	算术运算符
		%		求余（取模）

		++		只有变量才能使用。
				++a:先自加1，再进行其它运算。
				a++:先进行其它运算，再自加1。
		--		只有变量才能使用。
				--a:先自减1，再进行其它运算。
				a--:先进行其它运算，再自减1。

		+=		a++等同于a+=1，也等同于a=a+1
				字符串拼接也可以用 +=
		-=		a--等同于a-=1，也等同于a=a-1

关系运算符

		== 		- 等于
		===		- 值和类型都等于
		!= 		- 不等于
		!==		- 值和类型都不等于

		<  		- 小于
		<= 		- 小于或等于
		>  		- 大于
		>= 		- 大于或等于

逻辑运算符

		与 		或 		非
		&&	- 两个条件都成立，结果为 true
		||	- 两个条件只要有一个成立，则结果为 true
		! 	- 值取反。true 变成 false。false 变成 true。

运算优先级

		算术运算 > 位运算 > 关系运算 > 逻辑运算

位运算

		>>		右位移
		<<		左位移

特殊运算符

运算符 返回值 描述
delete true/false 从对象中删除一个属性，或者从数组中删除一个元素 
var obj = {a:1,b:2};
delete obj.a;
// 结果：obj == {b:2}

var arr = [1,2,3];
delete arr[0];
// 结果：arr == [2,3]
typeof string 返回参数的数据类型
void undefined 始终返回 undefined
isNaN true/false 检测指变量是否为非数字，即是否为 NaN
var a="5王尼玛"-5;
a == NaN 是错误滴
要用 isNaN(a) 来检测
instanceof true/false 检测指定对象是否是特定类的实例 
var arr = [1,2,3];
console.log(arr instanceof Array);
console.log(arr instanceof Object);
new object 创建一个新的对象实例 
var obj = {};
var o2 = new Object();

三元运算符（三目运算符）

		?:		三元运算符。

		如：var a = x>5 ? x : 5;

		if(x>5) a = x;
		else a = 5;

数据转换

		parseInt()		将字符串转换成整数类型
		parseFloat()	将字符串转换成浮点数（小数）类型
