Number - 数值型

		整型（整数） - int（integer）
		浮点型（小数） - float
		JavaScript 中所有数值都使用 浮点类型 存储。

		二进制：010 011
		十进制：10 200 50.2
		八进制：010 056 007
		十六进制：0xf 0x02 0xff
		RGB #FFFFFF rgb(255,255,255)

String - 字符串类型

		用单引号或者双引号包含的任意文本内容，称为“字符串”。
		字符串的起始符：起始、结束都是对应的相同的引号。

Boolean - 布尔类型

		布尔类型数据只能是 true 和 false 其一，代表“真的”和“假的”。
		布尔类型常用来作条件判断。

		0、undefined、null、NaN、""都被认为是 false。

		NaN - not a number
