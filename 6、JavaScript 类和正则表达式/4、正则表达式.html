正则表达式

		JavaScript中正则表达式对象名称为 RegExp，即“有规则的表达式”

		比如：/[a-z]+/
		以 a-z 之间的字母重复1次或者多次的字符串
		abcdefg   good   hello  abcd123

		/[a-z0-9]+/
		匹配任何以数字、小写字母为组合的字符串

		"Hello W!".replace("W","World")
		"Hello World!"

		正则表达式，就是通过一系列规则的标识符来描述一段有规律的字符串的表达式。
		而这些标识符，我们称之为“模式”

创建正则表达式

		字面量直接创建
		/[a-z]/ig
		一定记住，没有引号，它是一种字面量表达式，而不是“字符串”

		构造函数创建
		var reg = new RegExp("[a-z]", "ig");
		通过对象参数的形式来构建，有引号，字符串形式
		好处在于，能够在表达式中插入其它变量内容

		var start = "c";
		var end = "f";

		new RegExp("["+start+"-"+end+"]", "ig");
