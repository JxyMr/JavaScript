修改元素属性

		通过对象属性设置元素属性：
		（只针对标准属性，特殊属性名如 class 访问时修改成 className,
		for 访问时修改成 htmlFor）

		var a = document.getElementById("link");
		var href=a.href;//a["href"]

		-------------------------------------------------------

		通过方法设置属性：
		getAttribute、setAttribute
		var href = a.getAttribute("href");

		hasAttribute、removeAttribute
